picsum 占位图 API

通过 URL 即时生成 PNG 占位图，支持自定义尺寸、文本和背景色，适用于原型开发、文档示例与前端调试。

基础用法

路径格式：/:width/:height?，省略高度时默认正方形。

GET /300/200
GET /256

查询参数

支持 text 与 color 参数，color 使用十六进制颜色值。

GET /400/240?text=demo&color=%23ff6b6b

快速体验

点击示例： /320/180?text=hello&color=%230ea5e9

返回内容类型为 image/png，默认缓存头为 max-age=86400。
